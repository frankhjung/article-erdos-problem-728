# Article: A New Frontier in Mathematics: AI Solves Erdős Problem #728

Live site: <https://frankhjung.github.io/article-erdos-problem-728/>

This repository contains the source and build tooling for the article.

## Source

Primary source file: [consciousness.md](consciousness.md)

## Build (Make)

Requirements:

- [pandoc](https://pandoc.org/)
- a TeX engine for PDF output (e.g.,
  [xelatex](https://www.overleaf.com/learn/latex/XeLaTeX)

Targets:

- `make` → builds HTML and PDF into `public/` directory
- `make clean` → removes the `public/` directory

## Output

- HTML: `public/index.html`
- PDF: `public/erdos-problem-728.pdf`
